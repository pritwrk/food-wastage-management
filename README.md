## 📖 Project Overview

A full-stack data application built to tackle local food wastage by connecting **food providers** (restaurants, supermarkets, grocery stores) with **receivers** (NGOs, shelters, individuals).

The system enables real-time food listing management, claim tracking, and trend analysis through an interactive Streamlit interface backed by a MySQL database.

---

## ⚙️ Tech Stack

| Layer | Technology |
|---|---|
| Database | MySQL |
| Backend | Python, Pandas |
| Frontend | Streamlit, Plotly |
| Query Analysis | MySQL Workbench, 15+ SQL Queries |
